Course Description

This is an introduction to the architecture and organization of modern computer systems. Topics include processor design, number system representation and computer arithmetic, input/output, memory systems, and threading. C and assembly language are covered.

Other Course Info

Course Credits: 3
Availability: Fall Only
Prerequisites: CMSCI 120 Introduction to Computer Science I or Permission by Instructor
Course Fee: $50
Required for:

  • Computer Science Major
  • Cybersecurity Major
Optional for:
  • Computer Science Minor
  • Cybersecurity Minor
To register for this course, go to MyMount and add it to your course planning on the registration tab.

Key Points of this Class (What you should take away)

  • How numbers and other data are represented in computer systems
  • How digital logic circuits can represent Boolean functions
  • How sequential circuits can be used to build CPUs
  • What assembly code looks like and how you can use it to write simple functions
  • How memory is managed in a computer system

What Else Should Students Know About This Course

Staying engaged and keeping up with assignments is key to succeeding in this class. Computer architecture introduces concepts that can seem abstract at first, so reviewing your notes regularly, practicing with examples, and asking questions will help everything click into place.
Hands-on work, such as coding exercises and labs, is just as important as the lectures because it reinforces what you learn and helps you apply the material. Don’t forget to take advantage of office hours, this help page, and your classmates whenever you need clarification or extra support. Using all the resources available to you will make the course much more manageable and rewarding.
