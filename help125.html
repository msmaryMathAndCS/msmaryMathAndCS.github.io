What is CMSCI 125?

CMSCI 125 is the second introductory computer science class taught at Mount St. Mary's University, typically taken in the spring semester of freshmen year for computer science majors. CMSCI 125 is a continuation of CMSCI 120, the first introductory computer science class, in which 125 emphasizes the object-oriented paradigm. Students learn about class design, inheritance, input and output to files, and simple data structures.

This page will serve to better one's understanding of the contents of CMSCI 125. The resources and tips listed below were researched by former/current CMSCI students and instructors.

What to Expect in CMSCI 125

  • Languages and Tools: CMSCI 125 focuses almost entirely on Java. Some instructors may reference Python occasionally, but the core of the course is Java programming. Most professors use Eclipse or VS Code as the primary IDE, and some assignments may also use online tools like JDoodle.

  • Online Resources: Professors and students reccomend using online references when office hours and the class textbook don't cut it . W3Schools and GeeksForGeeks both offer clear explanations and examples for Java concepts. The free online textbooks listed below are also excellent for reviewing topics at your own pace.

  • Skills to Gain: By the end of the course, students should feel comfortable writing simple Java programs. Students should also feel comfortable writing 30-60 lines of code to solve simple problems, organized into classes and methods. You’ll also build a strong foundation in object‑oriented programming, which is essential for almost all modern software development.

  • Real‑World Applications: Java remains one of the most widely used programming languages in industry, thanks to its robustness. The problem‑solving and algorithmic thinking skills developed in this class apply to nearly every area of computing, not just for computer science majors but many other majors such as cybersecurity and data science majors

  • Why This Class Matters: Even if you’re not a computer science major, CMSCI 125 teaches algorithmic thinking and problem‑solving skills that are valuable for any field. Understanding how computers interpret code also helps explain why programs behave the way they do. This is especially evident when programs don't work as intended

  • Where This Course Leads: CMSCI 125 is required for the Computer Science and Cybersecurity majors, as well as the CS minor. After completing this course, students typically move on to CMSCI 254 (Data Structures and Algorithms) and/or CMSCI 256 (Principles of Software Development).
